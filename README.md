# To-Do List API - Spring Boot

Projeto de API REST para gerenciamento de tarefas.

## Funcionalidades
- Criar tarefa (POST /tasks)
- Listar tarefas (GET /tasks)
- Buscar tarefa por ID (GET /tasks/{id})
- Deletar tarefa (DELETE /tasks/{id})

## Tecnologias utilizadas
- Java 21
- Spring Boot
- Spring Data JPA
- H2 Database
- Maven

## Como executar
1. Clonar o repositório
2. Abrir no IntelliJ
3. Rodar a aplicação
4. Acessar: http://localhost:8080/tasks
